# Inframaps

> A maps and location integration project for **inframotion GmbH**.

## About

Inframaps is a geolocation and mapping feature developed for inframotion GmbH. It integrates mapping services to display infrastructure project locations, sites, and routes for field teams and clients.

## Features

- Interactive map integration
- Infrastructure site visualization
- Location tracking and management
- Field team coordination support

## Getting Started

1. Clone the repository:
   git clone https://github.com/fuatsevinc/inframaps.git

2. Navigate into the project directory:
   cd inframaps

3. Open the project in your browser.
